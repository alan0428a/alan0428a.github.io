Getting Insight Into Smart Manufacturing!

Industry 4.0 is all about connectivity and data insight. The Smart Facotry Dashboard provide real-time statistics and the WIP process tracking system. Users can easily manages their factory while making decision backed by intensive data.

Real-Time Dashboard

The dashboard allows the user to see yield rate and volunmes easily. Statistics data are directly pushed and updated in realtime.

WIP Process Tracking

With the WIP Process Tracking system, the user can find the work-in-progess at a long production line with ease. Users only need to provide the serial number of the WIP, then the system will show manufacturing its process history.

Statistics Query

Need to check daily or hourly performance? No problem. Users can select the time range they are interested in and check the statistics directly.
